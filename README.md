# 🔐 Реализация алгоритма RSA на Python

Учебный проект, реализующий алгоритм асимметричного шифрования RSA с нуля — без использования сторонних криптографических библиотек.

---

## 📌 О проекте

Данная работа выполнена в рамках ознакомительной практики 

**Цель проекта:**  
Изучение математических основ алгоритма RSA, его практическая реализация на языке Python и анализ актуальных угроз безопасности.

📚 Математическая основа

Алгоритм RSA базируется на трёх ключевых математических понятиях:

    Простые числа — основа стойкости алгоритма

    Модульная арифметика — обеспечивает операции шифрования/расшифровки

    Функция Эйлера φ(n) — используется для вычисления секретной экспоненты

Корректность алгоритма гарантируется теоремой Эйлера:

    C^d ≡ (M^e)^d = M^(e·d) = M^(k·φ(n)+1) ≡ M (mod n)

⚠️ Анализ безопасности

В проекте рассмотрены следующие угрозы и уязвимости:

    Задача факторизации — основа криптостойкости RSA

    Короткие ключи — уязвимость ключей длиной до 1024 бит

    Атака Винера — при малой секретной экспоненте d

    Атака малой открытой экспоненты — при e = 3

    Side-channel атаки — утечка информации по времени/мощности

    Квантовые угрозы — алгоритм Шора

    MITM-атака — отсутствие встроенной аутентификации в RSA

Рекомендации: использование ключей ≥ 2048 бит, стандарта OAEP, инфраструктуры PKI.
